This collection contains curated prompts for the GPT-Image-2 interface. These prompts help you generate high quality images through the OpenAI API. Use these templates to save time while you build your vision.

### 🚀 How to use the prompts

The folder contains several text files categorized by style. Choose a file based on your project needs.

- **Portraits**: Use these for realistic human faces or fantasy creatures.
- **Posters**: These prompts help you design marketing materials or movie art.
- **UI Mockups**: Apply these when you need screen designs for applications.
- **Game Assets**: Use these for character sheets and environment backgrounds.

To use a prompt, open the text file with Notepad. Copy the text inside the file. Paste that text into the input box of the GPT-Image-2 interface. The system reads your instructions and generates a unique image based on the chosen prompt.

### 🛠 Troubleshooting

Computers sometimes block unknown files. If you find trouble opening the folder or viewing files:
- Ensure you extracted the ZIP file first. Running files inside the compressed folder causes errors.
- Check that your API credits remain active. If your account lacks balance, the image generation tool will not start.
- Update your web browser if the interface appears blank.

### 💡 Tips for better results

Improve your images by adding small details to the base prompts. 
- State the lighting mood (e.g., "sunny," "dim," "neon").
- Mention the art style (e.g., "oil painting," "digital render," "pencil sketch").
- Define the camera angle (e.g., "close up," "wide shot").

The system processes these tags to adjust the final output. Experiment with the order of your words to change the focus of the generation.
